## 关键说明
1. **Noto系列字体**：Google主导的开源字体，覆盖多语言，是Linux、Android等系统的主流选择，`Noto Sans CJK SC` 为简体中文字体变体，`Noto Serif CJK SC` 为宋体变体（部分文档场景默认）。
2. **字体别名**：Linux系统中常使用 `Sans`/`Serif`/`Monospace` 等通用别名，实际映射为具体字体（如Ubuntu下`Sans`映射为Ubuntu字体，Fedora下映射为Noto Sans）。
3. **国产定制化**：国产OS（如深度、统信UOS）或国产ROM会基于开源字体二次定制（如深度体、鸿蒙字体），默认字体可能略有差异。
4. **等宽字体**：以上为通用UI/文档字体，若需等宽字体（代码编辑），补充如下：
   - Windows：Consolas（英文）、等线 (DengXian) 等宽版（中文）
   - macOS：Menlo（英文）、苹方等宽 (PingFang SC Mono)（中文）
   - Linux：Noto Mono（英文）、Noto Sans Mono CJK SC（中文）
5. **版本差异**：同一系统不同版本可能调整默认字体（如macOS从Helvetica Neue切换到San Francisco，Ubuntu从Liberation Sans切换到Ubuntu字体）。

可根据实际使用的系统版本、桌面环境微调，若需某类小众系统/环境的字体信息，可补充说明后进一步完善。
